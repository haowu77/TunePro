A lightweight instrument tuner for iOS, written in Swift.

## Building

If you want to build `Tuner` using Xcode, you can follow these instructions.

```fish
git clone https://github.com/timdubbins/TunePro.git
cd TunePro
open TunePro.xcodeproj
```

Happy coding!
